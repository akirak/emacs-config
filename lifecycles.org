* Life Cycles of the Nix packages
** NixOS Containers
*** Build the container
#+begin_src sh
sudo nixos-rebuild build --flake .#container  
#+end_src
*** Create and start the container
#+begin_src sh
nixos-container create flake-test --flake .
#+end_src

#+begin_src sh
nixos-container start flake-test  
#+end_src
